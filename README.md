## 🛠️ System Capabilities Demonstrated

### 1. High-Fidelity Software-in-the-Loop (SIL) Simulation
* Integrated custom Python/C++ autonomous control loops with advanced flight physics engines to validate performance under severe environmental disturbances before moving to hardware.

### 2. Autonomous control & Safety
* Implemented real-time dynamic path-planning and obstacle avoidance algorithms, ensuring regulation and trajectory tracking while ensuring safety.

### 3. Multi-Agent Consensus & Coordinated Control
* Developed distributed control strategies enabling multi-UAV teams to achieve state consensus, maintain formations, and execute synchronized mission objectives seamlessly.

### 4. Embedded Systems
* Flight-tested a custom, homemade quadrotor platform to validate theoretical control laws (such as non-linear and robust frameworks) in real atmospheric conditions.
